Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Ну тут всё супер просто,
Итерируя по каналу в ренже, из-за того что канал не закрыт, мы ждём данных с канала
Фиксится добавлением close(ch) после итераций

```
